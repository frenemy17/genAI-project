# Project: Intelligent Exam Question Analysis & Agentic Assessment Design

## From Educational Analytics to Autonomous Assessment Design

### Project Overview
This project involves the design and implementation of an **AI-driven educational analytics system** that analyzes exam questions and evolves into an autonomous, agentic assessment design assistant.

- **Milestone 1:** Classical machine learning and NLP techniques applied to historical exam data and empirical student responses to predict empirical difficulty and identify cognitive load (Bloom's Taxonomy).
- **Milestone 2:** Extension into an agent-based AI application that autonomously reasons about assessment gaps, retrieves curriculum best practices (RAG), and plans balanced examination strategies.

### Milestones & Deliverables

#### Milestone 1: Educational Analytics Pipeline (Mid-Sem)
**Objective:** Predict cognitive load and empirical difficulty using historical student behavioral data and question text, focusing strictly on classical ML pipelines *without LLMs*.

**Key Deliverables:**
- Problem understanding & Educational context.
- System architecture diagram.
- Working deployed application with detailed visual UI (Streamlit Cloud).
- Model performance evaluation report (Accuracy >91%, VotingClassifiers).

#### Milestone 2: Agentic Autonomous Assessment Assistant (End-Sem)
**Objective:** Extend the system into an agentic pedagogical strategist that reasons about assessment balance and retrieves testing best practices to generate structured exam recommendations.

**Key Deliverables:**
- **Publicly deployed application** (Link required).
- Agent workflow documentation (States & Nodes).
- Structured assessment gap reports generation.
- GitHub Repository & Complete Codebase.
- Demo Video (Max 5 mins).
